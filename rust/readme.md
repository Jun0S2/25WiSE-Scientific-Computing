# How to run Rust with Cargo

cargo run --bin 2557_Hello_World

```bash
rust/
├── Cargo.toml
└── src/
    └── bin/
        └── 2557_Hello_World.rs
```

Note ) chmod는 필요 없음
chmod +799처럼 권한을 바꾸는 건 실행 파일(.exe, .out)일 때만 필요.
